# yoloket
온라인 쇼핑몰에서 상품의 사이즈/색상을 선택하는 화면을 만들어봅니다.

# 화면 구성
상품의 사이즈/색상을 선택하는 것을 필수 기능으로 구현하되,
보다 확장성이 좋은 기능을 만들기 위해, 제품 선택의 플로우를 알아야한다고 생각하여,
제품의 목록을 보여주는 메인 화면과 상세 화면을 모두 만들어봅니다.
